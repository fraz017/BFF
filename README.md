# README


* Ruby version: 2.4.2

* Rails version: 5.0.1

* Configuration: 
  
  ->  bundle install

* Database creation:
  
  -> rake db:create
  -> rake db:migrate
  -> rake db:seed

* How to run the server

  -> rails server

* Deployment instructions

  We are using capistrano for deployment..

  -> cap production deploy
